### 命令行操作光标常用命令

|      命令      |        含义        |
| :------------: | :----------------: |
| ```ctrl + a``` |     跳转到行首     |
| ```ctrl + e``` |     跳转到行尾     |
| ```ctrl + l``` |        清屏        |
| ```ctrl + y``` |   回复上一步操作   |
| ```ctrl + k``` | 删除光标后面的文字 |
|  ```ctrl+u```  | 删除光标前面的文字 |
|                |                    |
|                |                    |
|                |                    |
